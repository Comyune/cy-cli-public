# Usage
<!-- usage -->
```sh-session
$ npm install -g cy-cli
$ cy COMMAND
running command...
$ cy (-v|--version|version)
cy-cli/0.0.1 darwin-x64 node-v14.5.0
$ cy --help [COMMAND]
USAGE
  $ cy COMMAND
...
```
